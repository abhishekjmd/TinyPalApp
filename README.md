## 🚀 How to Use the App

Follow these steps to set up and run the app locally:

### 1️⃣ Install Dependencies

From the project root directory, run:

```bash
# Using npm
npm install

# OR using Yarn
yarn install
````

---

### 2️⃣ Start the Metro Bundler

Run the Metro server to bundle and serve the app:

```bash
# Using npm
npm start

# OR using Yarn
yarn start
```

---

### 3️⃣ Run the App on Your Device or Emulator

#### 🟢 For Android:

```bash
npm run android
```

#### 🍎 For iOS:

```bash
npm run ios
```

If everything is configured correctly, the app will open on your Android Emulator, iOS Simulator, or a connected physical device.
